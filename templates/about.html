About This Project

This Crop Recommendation System helps farmers and agriculture professionals determine the most suitable crop to cultivate based on various soil and weather parameters.

Users can input values such as Nitrogen (N), Phosphorous (P), Potassium (K), pH level, temperature, humidity, and rainfall to get a machine learning-based prediction for the best crop to grow.

The system is powered by a trained classification model and deployed using Flask. It offers a user-friendly interface with multiple pages including prediction, crop info, history tracking, and this about section.
